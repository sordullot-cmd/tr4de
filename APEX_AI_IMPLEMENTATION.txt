🔧 INSTALLATION - PROCHAINES ÉTAPES

═══════════════════════════════════════════════════════════════

ÉTAPE 1: Installer packages
──────────────────────────
$ npm install @supabase/supabase-js @supabase/ssr ai openai zod

ÉTAPE 2: Créer schéma Supabase
──────────────────────────────
Supabase → SQL Editor
→ Copier contenu supabase/schema.sql
→ Exécuter

ÉTAPE 3: Créer Storage bucket
──────────────────────────────
Supabase → Storage
→ New Bucket: "trade_screenshots"
→ Make Public

ÉTAPE 4: Ajouter fonction SQL
──────────────────────────────
Supabase → SQL Editor
→ Copier fonction search_trades_by_embedding() depuis APEX_AI_SETUP.md

ÉTAPE 5: Configuration .env.local
─────────────────────────────────
Copier .env.local.example → .env.local
Remplir les 4 variables:
  NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
  NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...
  SUPABASE_SERVICE_ROLE_KEY=eyJ...
  OPENAI_API_KEY=sk-proj-...

ÉTAPE 6: Tester
───────────────
$ npm run dev
→ Aller sur sidebar: 🤖 Agent IA
→ Vérifier 3 onglets s'affichent
→ Tester ajouter un trade
→ Poser une question au chat
