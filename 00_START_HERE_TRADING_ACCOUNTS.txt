## 🚀 COMMENT PROCÉDER

### ÉTAPE 1 : Exécuter la Migration (5 min)
1. Va dans Supabase Dashboard
2. Ouvre "SQL Editor"
3. Copie le contenu de: supabase/migrations/001_create_trading_accounts.sql
4. Clique "Run"
5. ✅ Done - La base de données est prête

### ÉTAPE 2 : Intégrer dans le Code (10 min)
SUIS **INTEGRATION_GUIDE.md** pour modifier :
- components/DashboardNew.jsx (4 modifications)
- components/TradeForm.tsx (3 modifications)

### ÉTAPE 3 : Tester (5 min)
1. Crée un compte dans "Comptes de Trading"
2. Ajoute un trade - doit sélectionner le compte
3. Vérifie que le trade est lié au bon compte

## 🔒 SÉCURITÉ

- Chaque utilisateur voit UNIQUEMENT ses propres comptes
- RLS (Row Level Security) activée
- Suppression d'un compte = suppression de tous ses trades
- Authentification requise
- Impact: ✅ ZÉRO sur les fonctionnalités existantes
