# ApexTrader 🎯

Un **dashboard de trading avec intégration multi-brokers** pour visualizer vos vrais trades.

## ✨ Features

- 📊 Dashboard professionnel avec stats réelles
- 🔗 Support multi-brokers (Tradovate, Vantage MT5)
- 📈 Données en temps réel depuis vos comptes brokers
- 🔒 Credentials sécurisés côté serveur
- 🚀 Architecture prêt pour scalabilité

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Puis connectez votre broker :
1. Ouvrez http://localhost:3000
2. Cliquez **"⚡ Connect Broker"**
3. Entrez vos credentials
