# Archived Files

Ce dossier contient les fichiers non utilisés ou obsolètes du projet tr4de.

## Structure

### `/documentation/`
Tous les fichiers markdown de documentation et guides (APEXAGENT_*.md, COACH_MODE_*.md, etc.)
Ces fichiers sont conservés pour référence mais ne sont pas actifs dans le projet.

### `/old-components/`
Anciennes composantes React/JSX du dossier `app/components/`
Inclut les versions anciennes de Dashboard.jsx, DashboardNew.jsx, etc.
Utilisées lors du développement mais remplacées par les versions du dossier `components/` racine.

### `/tests-old/`
Anciens fichiers de test (test-*.js, test-*.mjs)
Conservés pour référence mais remplacés par les tests actuels.

### `/config-old/`
Fichiers de configuration/script obsolètes:
- `mt5_api.py` - API MT5 ancienne
- `start-mt5-api.bat` - Script de démarrage ancien
- `QUICK_REFERENCE.sh` - Référence rapide obsolète

## Note

Si vous avez besoin de récupérer un fichier, il se trouve dans le sous-dossier correspondant. Il suffit de le déplacer vers le répertoire approprié du projet.
